# Data Card

## Dataset
DataCo Supply Chain Dataset

## Source
Kaggle – DataCo Supply Chain Dataset

## Problem
Predict whether an order is likely to be delivered late using only information available before shipment.

## Target Variable
Late Delivery Risk

- 1 = Late Delivery
- 0 = On-time Delivery

## Number of Records
180,519

## Number of Features
46

## Data Cleaning
- Removed unnecessary columns
- Encoded categorical variables
- Handled missing values
- Split into training and testing datasets
